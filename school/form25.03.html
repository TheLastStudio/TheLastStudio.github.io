  1. Підпрограма - невелика програма, що виконує частину роботи в загальній програмі. В Delphi є процедури і функції (procedure і function). Процедури - закінчені підпрограми, які нічого не повертають, функції - підпрограми, які обробляють дані у ході роботи основної програми, повертає значення певного типу.
  2. Procedure:

procedure name (param1 : type1 ..., paramN : typeN);

var

// variables

begin

//code

end;

name(param1 ..., paramN); // call

Function:

function name (param1 : type1 ..., paramN : typeN) : type;

var

// variables

begin

//code

name := value; // returning value

end;

variable := name(param1 ..., paramN); // call

  1. Фактичні параметри вказуються під час виклику підпрограми, формальні вказуються під час оголошення підпрограми.
  2. Параметри-значення param : type передаються в підпрограму як нові локальні змінні, тоді як параметри-змінні var param : type передаються як посилання на існуючі змінні і всі зміни цього параметра зберігаються після завершення підпрограми.
  3. Масив - впорядкований набір змінних одного типу. Кожна змінна має свій порядковий номер - індекс. Визначається масив у блоці var, як name:array[first_index..last_index] of type; Ініціалізується кожен елемент масиву як name[i] := value. Можна створювати багатовимірні масиви, наприклад
    array[1..5] of array[1..5] of longint або array[1..5, 1..5] of longint.
  4. Якщо ми будемо вважати “правим” той трикутник, який називається “верхнім”, то ось процедура:

procedure sup_of_upper(matrix: array of array of longint; len: longint; var result: longint);

var

  i, j: longword;

begin

  result := 0;

  for i := 1 to len do

  begin

    for j := i to len do

    begin

      result := result + matrix[i][j];

    end;

  end;

end;

program FindSymmetricNumbers;

const

  Maxsize = 100;

var

  input_file: Text;

  Numbers: array[1..Maxsize, 1..Maxsize] of longint;

  size, i, j, k: longint;

  SymmetricNumbers: array[1..Maxsize*Maxsize] of longint;

  s_count: longint;

function IsSymmetric(Number: longint): boolean;

var

  s: String;

  i, j: longint;

begin

  s := IntToStr(Number);

  j := length(s);

  for i := 1 to length(s) div 2 do

  begin

    if s[i] <> s[j] then

    begin

      IsSymmetric := False;

      exit;

    end;

    dec(j);

  end;

  IsSymmetric := True;

end;

begin

  assignfile(input_file, 'input.txt');

  reset(input_file);

  readln(input_file, size);

  for i := 1 to size do

  begin

    for j := 1 to size do

    begin

      read(input_file, Numbers[i, j]);

    end;

  end;

  closefile(input_file);

  s_count := 0;

  for i := 1 to size do

  begin

    for j := 1 to size do

    begin

      if IsSymmetric(Numbers[i, j]) then

      begin

        inc(s_count);

        SymmetricNumbers[s_count] := Numbers[i, j];

      end;

    end;

  end;

  if s_count > 0 then

  begin

    writeln('Symmetric Numbers:');

    for k := 1 to s_count do

    begin

      writeln(SymmetricNumbers[k]);

    end;

  end

  else

  begin

    writeln('No symmetric numbers found.');

  end;

end.

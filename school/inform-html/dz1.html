Сайт про те, як працює цей сайт

Тут я розкажу про те, як зробив цей сайт, стилі, шрифт, розмітку і звісно цей чудовий фон!

Почну з шрифту та кольорів:

  • Знаходимо на Goole Fonts гарний шрифт і натискаємо на нього
  • На панелі Styles знаходимо Select this style і натискаємо
  • Код з відкритого віконечка, що починаеться з link копіюємо у head
  • Код з font-family вставляємо у body style="сюди"

    • Тепер ми маємо жахливий шрифт, бо іншого ми знайти не могли новий чудовий шрифт.

  • Білий текст це просто text="#eee" у тегу body
  • Для такого тексту існує гарний сайт, де достатньо просто ввести текст і натиснути "скопіювати HTML". Цей сайт просто генерує коди кольрів і робить за вас нудну роботу по запиханню вставлянню кожної букви у тег span та додаванню параметру style="color:який небудь колір"

У тегу head у мене є додатковий тег style. У ньому можна прописати стилі css для того, щоб вони додалися до кожного відповідного тегу. У моєму випадку там є параметри для посилань (тег а), блоків коду та абзаців - щоб були переноси тексту.

Також там є ще декілька параметрів. Ось вони:

        ::-webkit-scrollbar {
          width: 8px;
        }
        ::-webkit-scrollbar-track {
          background: #111;
        }
        ::-webkit-scrollbar-thumb {
          background: #070;
          border-radius: 4px;
        }
        ::-webkit-scrollbar-thumb:hover {
          background: #050;
        }

Завдяки цьому коду, зправа у вас не жахливий сірий прямокутник зі стрілками, а гарна тоненька лінія прокрутки.

І мабуть найцікавіше - це задній фон!

            var q = document.getElementById('q');
            var red = 0;
            var green = 255;
            var blue = 0;
            w=q.width=$(document).width();
            h=q.height=$(document).height();
            p=Array(256).join(1).split('');
            c=q.getContext('2d');
            c.fillStyle='rgba(0,0,0,1)';
            m=Math;
            setInterval(function(){
                c.fillStyle='rgba(0,0,0,0.08)';
                c.fillRect(0,0,w,h);
                c.fillStyle='rgba('+red+','+green+','+blue+',1)';
                p=p.map(function(v,i){
                    r=m.random();
                    c.fillText(String.fromCharCode(m.floor(2720+r*33)),i*10,v);
                    v+=10;
                    return v>768+r*1e4?0:v
                })
            },33)

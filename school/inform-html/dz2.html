Сайт про списки

Списки бувають:

Нумерованими

  1. Створюэться за допомогою тегу ol.
  2. Кожен елемент починається з тегу li.
  3. За замовчуванням нумерація іде з 1, арабськими цифрами.

Атрибути:

  1. Початок нумерації задається атрибутом start.
  2. Всередині можна використовувати будь-які теги тексту.
  3. Тип нумерації задається атрибутом type.
  4. Атрибут type можна прописувати як для всього списку, так і для окремих елементів.

Маркованими

  • Створюэться за допомогою тегу ul.
  • Кожен елемент починається з тегу li.
  • За замовчуванням використовується символ •.

Атрибути:

  • Тип маркування задається атрибутом type.
  • Всередині можна використовувати будь-які теги тексту.
  • Існують типи disk, circle, square.
  • Атрибут type можна прописувати як для всього списку, так і для окремих елементів.
